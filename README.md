3️⃣ College Event Feedback Analysis (Future Interns Data Science and Analytics Internship Task 2)

## Overview
This project focuses on analyzing student feedback to understand overall satisfaction and areas of improvement. The dataset was cleaned and preprocessed using Python, validation of rating values and sentiment analysis of ratings using basic NLP techniques. The processed data was then visualized using Google Looker Studio to build an interactive dashboard.

## Features
Cleans the data. Validates and analyzes rating data (1–5 scale). Performs sentiment analysis on textual feedback (Positive / Neutral / Negative). Interactive dashboard built in Google Looker Studio.
Project Files
data.csv first.py

## Dashboard Features
KPI cards (Total feedback responses, Overall average rating, Positive student feedback, Negative student feedback)
Visualzations (Ratings by Aspect, Sentiment Distribution, Rating Distribution, Heatmap of Ratings)
Actionable Reccomendations

## How to Use this Project
Run the Python script to clean and preprocess the raw feedback CSV file.
Review the processed dataset containing validated ratings and sentiment labels.
Analyze insights to suggest improvements for future college events.
